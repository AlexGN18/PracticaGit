# Documentos estructurados

A veces es útil tener diferentes niveles de encabezados para estructurar sus documentos. Comience líneas con un `#` para crear encabezados. Múltiples `##` en una fila denotan tamaños de encabezado más pequeños.

### Este es un título de tercer nivel

Puede usar un `#` hasta llegar a `######` seis para diferentes tamaños de encabezado.

Si desea citar a alguien, use el caracter> antes de la línea:

> Café. La mejor suspensión orgánica que se haya ideado jamás ... Derroté a los Borg con ella.
> - Capitán Janeway
